助念申請電話接洽流程

1 接聽電話：基礎確認

1.1 招呼與身分確認：

阿彌陀佛，這裡是淨土宗竹山念佛會，
我是某某（居士）。
請問怎麼稱呼您？

目的：
先禮貌問候，並確認來電者的基本資訊，方便後續稱呼與溝通。

1.2 詢問介紹來源：

請問是誰跟您介紹申請我們的助念？
或是您怎麼得知的呢？

目的：
了解家屬申請助念的管道或介紹人，方便後續回饋或管理。

1.3 了解申請人與往生菩薩的關係：

請問往生菩薩是您的哪位親人？（父母、配偶、子女、兄弟姊妹等）

目的：
確認申請人是否為家屬，並掌握與往生菩薩之間的親屬關係，利於後續安排。

2 往生菩薩基礎資訊

2.1 往生菩薩姓名與過世時間：

往生菩薩的大名？何時往生？是否在往生後八小時內？（國曆年月日時）

目的：
掌握往生菩薩基本身分與過世時點，做為安排助念時程的最重要參考。

2.2 往生地點與遺體現況：

請問往生菩薩是在家中、醫院還是其他地方往生？
是否還在原地，或已送至殯儀館？

目的：
瞭解遺體所在地，方便確定助念團隊前往地點與準備方式。

3 申請與家屬配合事項

3.1 服務原則與家屬參與：

我們助念是完全免費的，但有幾點希望家屬可以配合：

  1. 助念的過程，家屬必須一同參與念佛，這點是最重要的，往生菩薩聽到熟悉的聲音，就像有人牽他的手，這樣對往生菩薩更具助益，請問您們能配合嗎？
  2. 我們在助念的過程，請將香移除或至通風處處理，現場不要焚香，因為現在的香大多為化學香，對身體有害，請問您們能配合嗎？勸導
  3. 依《地藏經》所說，大多數的人，往生後七七四十九天，都會隨著業力跟因緣，確定下一輩子去哪裡。因此剛往生的四十九天，最需要幫忙，每一天都是關鍵，不是只有第七天。我們念佛會，每週日都會舉辦「念佛超薦法會」，這也是免費的。這段時間，請您們回來參加法會，跟隨法師與大眾一起共修念佛，借助大眾的力量，幫助我們的菩薩順利往生。勸導
  4. 為了鼓勵大眾發心助念，我們可能會拍照上傳至 LINE 或臉書，但不會拍攝到家屬或往生菩薩。請問您能接受嗎？

目的：
先向家屬說明助念服務的前提與條件，確保家屬理解並願意配合（包括一起念佛、不焚化學香、照片分享方式，以及做七期間回到道場參與超薦法會）。

3.2 助念過程所需物品：

助念時所需的物品，我們會準備，你們不用擔心，只需要準備椅子給我們蓮友坐即可。

目的：
讓家屬知道不收取任何費用並會自行攜帶法器、念佛機等，只需家屬協助準備座椅。

3.3 往生菩薩的生前狀況與需求：

助念的過程當中，我們會有簡略的開示勸勉往生菩薩，所以希望能對往生菩薩多了解一些：

  1. 往生菩薩過世的原因？或簡述一下最後的階段。
  2. 生前是否有念佛或宗教信仰？
  3. 有沒有特別的牽掛或未了心願，需要我們在助念時特別留意？

目的：
進一步瞭解往生菩薩的背景、病程或牽掛，便於助念時的開示與安慰引導。

3.4 家庭成員概況：

家裡有多少成員會一起參與助念？
是否有需要特別安撫的長者或親屬？

目的：
評估現場可能有多少人參與與空間需求，同時確認是否有需要特別關懷的家屬。

4 助念細節安排

4.1 助念時間與地點：

您希望申請助念的時段與地點？

目的：
明確確定助念的開始時間及地點，方便念佛會安排。

4.2 確認主要聯絡方式：

若助念的時間安排好了，要如何聯繫您？方便留電話或 LINE 嗎？

目的：
留下家屬的可靠聯絡方式，以便後續通知或臨時變更討論。

4.3 其他補充事項：

目前先根據以上重點安排。若有任何狀況或需求，可再與我們聯繫。

目的：
開放家屬提出其他需求或特殊情況，讓助念團隊有更充裕的準備。

5 通話簡要總結

好的，我們大概了解了，會盡速安排助念。
若有需要，可再打電話來與我們聯繫。阿彌陀佛！

目的：
通話結束前簡單總結與確認，讓家屬知道下一步，並保留回溝通的空間。

助念申請表（填寫範本）

1 申請人（家屬）資訊

  • 姓名（稱呼）
  • 聯絡電話 / LINE
  • 與往生菩薩關係
  • 介紹人 / 申請管道

目的：
確認申請人身分與往生菩薩的關係，並記錄申請來源便於聯繫追蹤。

往生菩薩基本資料

  • 姓名（性別）
  • 往生時間（國曆年月日時）
  • 往生地點（自宅 / 醫院 / 其他）
  • 遺體現況（殯儀館 / 自宅）

目的：
充分掌握往生菩薩的基本身分及目前遺體的安置情形，以利助念安排。

3 往生菩薩生前狀況

  • 過世原因（疾病 / 意外 / 其他）
  • 生前是否有念佛或特定信仰
  • 特別牽掛或未了心願

目的：
瞭解往生菩薩的生前境況，助於助念時的開示方向與心理安撫。

4 助念安排

  • 是否在往生後八小時內
  • 希望助念的時間
  • 助念地點（住家 / 醫院 / 殯儀館 / 其他）
  • 家庭成員參與人數概況

目的：
落實助念的時間、地點以及家屬參與度，讓助念團隊可進行人力配置與交通及時間安排。
